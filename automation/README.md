# Dentzy Appium Automation

This folder contains a pytest + Appium automation harness for the Dentzy Flutter app.

## Run

1. Start Appium server:
   `appium`
2. Run tests:
   `.venv\Scripts\python.exe -m pytest automation/tests -v`
3. Open the consolidated report at:
   `automation/reports/Final_Test_Report.xlsx`
